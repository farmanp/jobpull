Use this file as a short bridge if you land on GitHub first. The docs site is the source of truth for detailed setup and customization.

## At a glance

- start with a starter pack
- tune the role filters
- add or remove sources
- adjust cron and stale-job behavior
- choose Cloudflare or Docker for deployment

For a new board, the practical order is: `packs apply`, `sources`, then a crawl.
